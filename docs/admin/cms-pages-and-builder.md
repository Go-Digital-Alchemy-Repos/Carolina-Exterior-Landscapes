# CMS Pages And Builder

CMS pages use structured blocks instead of freeform HTML. Editors can create pages, schedule publishing, set per-page SEO metadata, choose a template, and compose content with retained generic blocks.

Retained block families include hero, rich text, text/image, cards, call-to-action, FAQ, section header, and form embed blocks.

Retired blocks for directory browsing, provider listings, applications, events, posts, recordings, newsletters, memberships, subscriptions, products, and payments have been removed.

# Getting Started

This admin area manages the retained website CMS: pages, reusable sections, menus, sidebars, media, forms, SEO, redirects, users, settings, documentation, and backups.

Use `/setup` to create the first administrator account. After setup, use `/auth/login` for direct backend access.

Public website content should be created through CMS pages and blocks. Retired starter modules are intentionally not available in the admin.
